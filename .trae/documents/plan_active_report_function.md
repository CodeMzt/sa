# 计划：实现RobStride电机主动上报功能

## 任务概述
根据EL05电机手册(PDF文档)，实现正确的主动上报功能，替代当前不正确的方法。

## 分析阶段
1. 从EL05.pdf文档中提取主动上报功能的正确通信协议
2. 理解主动上报的帧格式和参数设置方法
3. 对比当前代码实现与手册规范的差异

## 实现阶段
1. 创建新的主动上报控制函数
   - 开启主动上报函数
   - 关闭主动上报函数
   - 设置上报间隔函数
2. 修改robstride_motor.h添加函数声明
3. 修改robstride_motor.c添加函数实现
4. 更新can_comms_entry.c中的主动上报测试代码

## 测试验证
1. 替换现有的主动上报测试代码
2. 验证主动上报数据是否正确变化
3. 确保功能符合EL05手册规范

## 预期结果
- 主动上报功能按照EL05手册正确实现
- 电机能够按设定间隔主动发送反馈数据
- 测试时能够观察到实时变化的数据
